## Page 1: System Overview & Introduction

### 1.1 Project Background
The ITUM Event Management System is a comprehensive web-based platform designed to modernize how students at the Institute of Technology University of Moratuwa discover, register for, and manage academic and extracurricular events. Traditional paper-based or manual event management systems often lead to inefficiencies, missed opportunities, and administrative overhead. This digital solution addresses these challenges by providing a centralized, user-friendly platform.

### 1.2 Problem Statement
Before this system:
- Students lacked a centralized platform for event information
- Manual registration processes were time-consuming
- Event organizers struggled with attendance tracking
- No real-time updates on event availability
- Difficulty in preventing duplicate registrations

### 1.3 Solution Overview
Our system provides:
- **Centralized Event Directory**: All university events in one place
- **One-Click Registration**: Streamlined sign-up process
- **Real-time Updates**: Instant confirmation and status
- **User Management**: Secure authentication and profiles
- **Mobile Responsive**: Accessible on all devices

### 1.4 Technical Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP 7.4+
- **Database**: MySQL 5.7+
- **Server**: Apache HTTP Server
- **Protocols**: HTTP, RESTful APIs

### 1.5 Target Users
- ITUM Students (primary users)
- Event Organizers
- University Administration
- Faculty Members
